# Contributing

1. Use Node.js 20 or newer.
2. Run `npm ci`.
3. Keep Plannotator integration on its documented public event API.
4. Do not weaken mandatory approval, second-warning overrides, safe Git behavior, the three-repair ceiling, final-only tab reopening, or sole-writer policy.
5. Run `npm run verify` and `npm run test:compat`.
6. Inspect `npm pack --dry-run` before release.

Do not update the adapted design-guidance snapshot without intentionally regenerating its manifest and reviewing third-party licensing.
